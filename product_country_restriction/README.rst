This module allows to define several rules to restrict product flows based on
countries or country groups (Embargoes).

It provides rules that apply:

* Globally (for all products)
* Per Category
* Per Product
* Per Variant

Moreover, these rules can be set with date windows.

**Table of contents**

.. contents::
   :local:

Usage
=====

This module cannot be used alone.

It defines method that helps to define if product can be used in a particular
country or not for a given date (today by default).
