This Odoo module allows you to set allowed pricelists for customers
and restrict the selection of pricelists to only those allowed for
the partner in the sale order.

.. IMPORTANT::
   This is an alpha version, the data model and design can change at any time without warning.
   Only for development or testing purpose, do not use in production.
   `More details on development status <https://odoo-community.org/page/development-status>`_

Configuration
=============

1. Go to Sales ‣ Configuration ‣ Settings.
2. In the Pricing section, check the box next to Use Partner Pricelist and Save.

Usage
=====

1. Go to the Customers menu.
2. Select the customer/partner for whom you want to set the allowed pricelists.
3. Define the Allowed Pricelists field and select the pricelists that should be allowed for this customer. If you want to allow all pricelists, leave the field empty.
4. Save the changes.
